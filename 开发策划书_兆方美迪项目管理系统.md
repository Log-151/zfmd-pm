**文档版本**：v1.0

**适用对象**：兆方美迪（甲方）内部运营管理团队

---

## 一、项目背景与目标

兆方美迪作为新能源领域风电光伏数值天气预报服务商，业务覆盖合同签署、项目开工、发票开具、回款跟踪、数值天气服务续期等多个环节。现阶段各业务数据分散于多张 Excel 表格中，存在以下痛点：

- 数据孤岛：各模块表格相互独立，缺乏关联，无法全局统览
- 效率低下：手工统计、重复录入，容易出错
- 预警缺失：回款逾期、天气服务到期等关键风险无法及时提示
- 汇报困难：缺少可视化统计报表，管理层难以快速掌握经营全貌

本项目目标是将现有 Excel 业务流程迁移至一套 **Web 端业务管理系统**，实现数据集中管理、流程数字化、风险自动预警、经营数据可视化分析。

---

## 二、系统总体架构

| 层次 | 技术选型 | 说明 |
|------|---------|------|
| 前端 | React + Vite + TailwindCSS | 现代响应式 Web 界面 |
| 后端 | Node.js + Express | RESTful API 服务 |
| 数据库 | PostgreSQL | 企业级关系型数据库，数据安全可靠 |
| 部署 | 云端服务器（Web 应用） | 浏览器直接访问，无需安装客户端 |

系统采用 B/S 架构，员工通过浏览器即可访问，数据统一存储于服务器，无需本地安装任何软件。

## 四、通用功能

以下功能在所有业务模块中统一支持：

| 功能 | 说明 |
|------|------|
| 登录认证 | 账号密码登录，会话有效期 7 天，退出登录立即失效 |
| 关键词搜索 | 每个模块顶部均有搜索框，支持按客户名称、编号等快速检索 |
| 多维筛选 | 下拉筛选省份、销售经理、状态等维度，支持组合条件 |
| 排序 | 支持按金额、日期等字段排序 |
| 批量 CSV 导入 | 上传 CSV 文件后自动解析，支持将历史 Excel 数据一次性迁移 |
| 单条录入 | 表单填写新增单条记录 |
| 编辑 / 删除 | 每条记录均可编辑或删除，删除带二次确认 |
| CSV 导出 | 将当前筛选结果导出为 UTF-8 BOM 编码 CSV，Excel 可直接打开 |
| 自定义字段 | 每个模块支持管理员自定义扩展字段，灵活适配业务变化 |
| 一键备份 | 侧边栏提供"一键备份全量数据"按钮，下载包含全部6个模块数据的 ZIP 压缩包 |
