## 三、数据模型设计

### 3.1 核心实体关系

```
┌─────────────┐
│   客户      │
│  (Customer) │
└──────┬──────┘
       │ 1:N
       ▼
┌─────────────┐      1:1      ┌─────────────┐
│   合同      │──────────────→│  开工申请   │
│  (Contract) │               │(WorkStart)  │
└──────┬──────┘               └─────────────┘
       │ 1:N
       ├──────────────────────────────┐
       ▼                              ▼
┌─────────────┐               ┌─────────────┐
│   发票      │               │  服务续约   │
│  (Invoice)  │               │(ServiceRen.)│
└──────┬──────┘               └─────────────┘
       │ 1:N
       ▼
┌─────────────┐      1:N      ┌─────────────┐
│  应收款     │──────────────→│   回款      │
│(Receivable) │               │  (Payment)  │
└─────────────┘               └─────────────┘
```

### 3.2 核心字段清单

#### 客户 (Customer)
```
- 客户编码（自动生成）
- 一级公司
- 二级公司
- 三级公司
- 客户名称
- 省（区）
- 集团
- 场站名称
- 场站类别（风电场/光伏电站）
- 场站容量
- 创建时间
- 更新时间
```

#### 合同 (Contract)
```
- 合同编号（主键）
- 开工申请编号（关联）
- 售后服务编号（关联）
- 合同变更号（关联）
- 客户名称（关联客户）
- 合同名称
- 省（区）/集团/场站名称
- 场站类别/容量
- 产品线（风电/光伏/数值天气预报）
- 合同项目内容
- 销售经理/销售联系人
- 合同存档日期
- 合同签订日期
- 服务收费起始/终止时间
- 初装费
- 预测服务费
- 合同总额（含税）
- 不含税合同金额
- 交付部门
- 项目经理
- 合同交底会时间
- 合同状态（执行中/已完成/已终止）
```

#### 发票 (Invoice)
```
- 发票编号（主键）
- 合同号（关联）
- 开票日期
- 申请开票日期
- 开票单位
- 发票金额（含税）
- 税率
- 不含税金额
- 承诺回款日期
- 承诺回款金额
- 实际回款日期
- 实际回款金额
- 发票状态（正常/作废）
- 作废日期
- 应收款预警状态
```

#### 回款 (Payment)
```
- 回款编号（主键）
- 合同号（关联）
- 回款日期
- 付款单位
- 省（区）/集团/场站名称
- 产品线
- 汇票回款金额
- 现金回款金额
- 回款总额
- 回款比例（自动计算）
- 款项名称
- 销售经理
- 关联发票（可多张）
```

#### 应收款 (Receivable)
```
- 应收款编号（主键）
- 合同号（关联）
- 应收款项名称
- 应收款金额
- 应收时间
- 承诺回款时间
- 承诺回款金额
- 实际回款时间
- 实际回款金额
- 超期时间（月）（自动计算）
- 回款状态（未回款/已回款/已扣款/坏账）
- 到货时间
- 验收时间
- 进入应收日期（自动计算）
- 晚回款月数（自动计算）
```

#### 开工申请 (WorkStart)
```
- 开工申请编号（主键）
- 对应合同编号（关联）
- 开工变更申请表编号
- 开工申请取消时间
- 是否发生成本费用
- 成本费用处理
- 开工申请流转时间
- 省（区）/集团/场站名称
- 场站类型
- 产品线
- 开工项目内容
- 销售经理
- 项目交底会时间
- 预计合同金额
- 预计成本
- 实际合同金额
- 交付部门
- 项目经理
- 到货时间
- 验收时间
- 未回合同预警状态
```

#### 服务续约 (ServiceRenewal)
```
- 记录编号（主键）
- 合同号/开工号（关联）
- 场站名称
- 场站类别
- 续签前服务到期时间
- 续签后服务开始时间
- 中断时间（月）
- 中断期间服务费处理
- 到期预警状态（3 月/2 月/1 月/超期）
- 已停止预报时间
- 断档月份记录
```

| 售后服务预警 | 未回合同的售后服务单 | 系统通知 | ⭐⭐⭐⭐⭐ |
| 账龄超期预警 | 应收款超 30/60/90 天 | 系统通知 + 邮件 | ⭐⭐⭐⭐⭐ |
